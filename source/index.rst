.. container:: lead

	Telephone is a Discord bot for :t:`Splatoon 2`, themed around the :t:`Octo Expansion`.

.. warning::
	If you have not completed the entire Octo Expansion, be forewarned: there are spoilers in some of the commands.

.. note::
	Some functions, such as profiles and minigames require 'registration' first. You can do so with ``.profile create``. See the topic ":doc:`/overview/getting-started`" for an overview of using Telephone bot.
